# Wardrobe Photos

Upload your clothing photos here. The app will reference them automatically.

## How to upload

1. Click **Add file → Upload files** on this folder in GitHub
2. Drag in your photos (JPG, PNG, WebP)
3. Name them clearly, e.g. `uniqlo-white-oxford.jpg`
4. Commit directly to `main`

## Using photos in the app

In the **Edit Piece** modal, type the filename in the  
**"Or filename from wardrobe-photos/"** field — no need to upload again via the form.

Example: if you uploaded `navy-chinos.jpg`, type `navy-chinos.jpg` in that field.

## Tips

- Use lowercase filenames with hyphens, no spaces (e.g. `cos-grey-knit.jpg`)
- Shoot on a plain light background for the cleanest card look
- Portrait orientation (3:4) works best with the card layout
